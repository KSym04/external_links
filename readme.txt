== Description ==
This plugin automatically convert links to anchor tag and adds rel=&quot;nofollow&quot; and target=&quot;_blank&quot;, for all the external links of your website item post.


== Installation ==
1. Upload `insert_on_header_footer/` to the `/oc-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in Osclass
3. Sit back and enjoy


== Frequently Asked Questions ==
1. Q: Where's the admin settings?
A: Currently, There is no available configuration panel for this plugin.
